Media Queries

Las Media Queries son una herramienta fundamental en CSS para crear diseños responsivos. Permiten aplicar estilos distintos según las características del dispositivo o la ventana del navegador, como el ancho, alto, resolución o tipo de pantalla.

Sintaxis básica

La sintaxis de una media query es la siguiente: 

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

En este ejemplo, cuando la pantalla tenga un ancho máximo de 600px, se aplicará un fondo azul claro.

Tipos de Media Queries

  • max-width: Aplica estilos si el ancho de la ventana es menor o igual al valor definido.
  • min-width: Aplica estilos si el ancho de la ventana es mayor o igual al valor definido.
  • orientation: Aplica estilos según la orientación del dispositivo (portrait o landscape).
  • resolution: Aplica estilos según la densidad de píxeles de la pantalla.

Ejemplo práctico

Un uso típico de media queries es modificar el diseño de un menú o el tamaño de texto según el dispositivo: 

/* Celulares */
@media (max-width: 600px) {
    .menu-navegacion {
        flex-direction: column;
    }
    .texto-seccion {
        font-size: 14px;
    }
}

/* Tablets */
@media (min-width: 601px) and (max-width: 1024px) {
    .menu-navegacion {
        flex-direction: row;
    }
    .texto-seccion {
        font-size: 16px;
    }
}

/* Computadoras */
@media (min-width: 1025px) {
    .menu-navegacion {
        flex-direction: row;
    }
    .texto-seccion {
        font-size: 18px;
    }
}

Con estas reglas, el diseño de la página se adapta a celulares, tablets y computadoras, asegurando una experiencia de usuario óptima.
