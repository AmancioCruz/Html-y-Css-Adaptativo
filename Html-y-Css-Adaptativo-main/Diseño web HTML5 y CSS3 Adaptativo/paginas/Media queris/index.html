Media Queries

Diferentes resoluciones de pantallas según el dispositivo

Los rangos de resoluciones más comunes suelen ser: en celulares entre 360 px y 428 px de ancho, en tablets entre 768 px y 1024 px, en laptops entre 1366 px y 1920 px, y en PC de escritorio desde 1920 px hasta 3840 px (4K), siendo estos los valores de referencia más utilizados en diseño y desarrollo.

Dispositivos Móviles

Resoluciones típicas: 360px - 767px

Características: Pantallas pequeñas, interacción táctil, conectividad variable.

Tablets

Resoluciones típicas: 768px - 1024px

Características: Pantallas medianas, orientación portrait/landscape, principalmente táctiles.

Laptops

Resoluciones típicas: 1025px - 1440px

Características: Pantallas grandes, entrada por teclado y ratón, alta resolución.

Escritorio

Resoluciones típicas: 1441px+

Características: Pantallas muy grandes, alta resolución, múltiples dispositivos de entrada.

Media Query - ¿Qué es?

Las media queries son una técnica de CSS3 que permite adaptar la presentación de contenido a diferentes condiciones como la resolución de pantalla, orientación del dispositivo o capacidades del mismo. Nos permiten generar diferentes estilos, diseños y estructuras del contenido de la página según las características del dispositivo.

Las media queries se utilizan para:

  • Aplicar estilos condicionalmente utilizando reglas en CSS: @media o @import
  • Segmentar medios específicos para style, link, source y otros elementos HTML con el atributo media
  • Crear experiencias adaptativas que mejoren la usabilidad en diferentes dispositivos
  • Optimizar el diseño para diferentes orientaciones (horizontal/vertical)
  • Ajustar la experiencia según las preferencias del usuario (modo oscuro, reducir movimiento)
/* Sintaxis básica de una media query */ @media [tipo-de-medio] and (condición) { /* Estilos CSS que se aplicarán si se cumple la condición */ } /* Ejemplo: aplicar estilos solo en pantallas con ancho máximo de 768px */ @media screen and (max-width: 768px) { body { font-size: 14px; } .menu { display: none; } }

Puntos de Interrupción (Breakpoints) Comunes

Los breakpoints son los valores de width en los cuales cambiamos los estilos de nuestra página para adaptarse a diferentes dispositivos. Aunque cada proyecto puede requerir breakpoints específicos, estos son los más comunes:

Móviles pequeños: < 576px
Dispositivos móviles en orientación vertical
Móviles: 576px - 767px
Dispositivos móviles en orientación horizontal y phablets
Tablets: 768px - 1023px
Tablets en orientación vertical y dispositivos pequeños
Laptops: 1024px - 1199px
Portátiles y tablets en orientación horizontal
Escritorio: 1200px+
Monitores de escritorio y pantallas grandes

Ejemplos Prácticos de Media Queries

Cambiar diseño de columnas

.container { display: flex; flex-direction: column; } @media (min-width: 768px) { .container { flex-direction: row; } }

Ocultar elementos en móviles

.sidebar { display: block; } @media (max-width: 767px) { .sidebar { display: none; } }

Ajustar tamaño de texto

body { font-size: 14px; } @media (min-width: 992px) { body { font-size: 16px; } }

Modo oscuro

@media (prefers-color-scheme: dark) { body { background: #222; color: #fff; } }
