es una unidad de medida relativa al elemento raiz (root) es muy similar a la em, sin embargo depende del tama単o que le asignemos e el estilo en la clase html si en el html asigno un font-zise de 20px, 1rem seria 20px, 2rem serian 40px, esta unidad no depende del contenedor padre solo del contenedor raiz(html).
